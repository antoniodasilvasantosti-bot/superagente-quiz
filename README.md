# 🎯 Superagente Quiz — Negócios em Informática

Quiz interativo para uso em sala de aula com painel do professor em tempo real.

## ✨ Funcionalidades

- 100 perguntas de múltipla escolha sobre Negócios em Informática
- Painel do professor com tabela de alunos, acertos e pontuação em tempo real
- Sons e voz falada em português (acerto, erro, nova pergunta, tempo esgotado)
- Timer de 30 segundos por pergunta
- Ranking automático com medalhas 🥇🥈🥉
- Modo tela cheia para projetor
- Animações e partículas

## 🚀 Como usar

1. Acesse o [link do quiz](https://base44.app/api/apps/6a1e3bfee87e8aacec768a92/files/mp/public/6a1e3bfee87e8aacec768a92/d1bf08096_index.html)
2. **Professor**: clique em "Área do Professor" → senha `2509`
3. Crie uma sessão e compartilhe o código com os alunos
4. **Alunos**: clique em "Entrar no Quiz" → nome + código
5. O professor seleciona as perguntas e os alunos respondem em tempo real

## 🛠 Tecnologias

- HTML5 + CSS3 + JavaScript puro
- Web Audio API (sons)
- Web Speech API (voz em português)
- Base44 Backend (banco de dados e sessões)
